Types of operators:
- Arithmetic: `+ - * / % // **`
- Comparison: `== != > < >= <=`
- Logical: `and or not`
- Assignment: `= += -= *= /=`

**Tokens are the smallest unit of a program:**
1. Keywords: reserved words (`if`, `else`, `for`, `while`, etc.)
2. Identifiers: variable/function names
3. Literals: constant values (`10`, `"Hello"`, `3.14`)
4. Operators: symbols (`+ - * /`)
5. Punctuators: `: , () { } []`
